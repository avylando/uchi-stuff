# README #

Подсветка синтаксиса для файлов .graph в `Visual Studio Code`.

Порт подсветки для Atom, *by Arseny Kruglikov*.

## Установка ##
* Скопировать директорию **graph-syntax-vscode** в `~/.vscode/extensions`
* Перезапустить редактор
